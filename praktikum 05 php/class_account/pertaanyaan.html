3. Apa fungsi kata kunci berikut ini dalam program PHP

  • extends : untuk membuat class turunan
  • __construct : method yang yang di jalankan pertama kali pada saat sebuah class di jalankan.
  • parent::__construct : menjalankan __construct pada parent nya.
  • parent::cetak() : untuk mencetak pada class induk
  • $this->nama : adalah sebuah variabel yang merujuk pada Property/Method yang saat ini sedang di definisikan
  • $this->hello() : adalah sebuah variabel yang merujuk pada fungsi yang saat ini sedang di definisikan
